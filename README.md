# 💫 About Me:
Frontend developer who thinks the web should feel alive. I build modern, interactive experiences using React, Tailwind CSS, GSAP, Framer Motion, Three.js, and creative UI systems like Aceternity UI. I turn ideas into polished interfaces where motion isn't decoration — it's storytelling. Think smooth scrolling, immersive 3D scenes, and micro-interactions that feel intentional. Behind the visuals, I bring C++ problem-solving skills and a logical approach that keeps performance, structure, and maintainability in check. 🔍 Shipping projects consistently and looking for opportunities where creative frontend meets real product impact.
